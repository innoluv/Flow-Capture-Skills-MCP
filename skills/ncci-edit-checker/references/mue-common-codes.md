# Common MUE Values — Quick Reference

MUE = Medically Unlikely Edit. This is the maximum number of units CMS considers medically reasonable for a single CPT/HCPCS code per beneficiary per date of service.

**MAI = MUE Adjudication Indicator**
- `1` = Line Edit (per claim line)
- `2` = Date of Service Edit (across all lines on same DOS)
- `3` = Clinical Edit (based on clinical criteria)

> These values reflect commonly published CMS MUE data. Always verify against the current CMS MUE tables at: https://www.cms.gov/medicare/coding-billing/national-correct-coding-initiative-ncci-edits/medicare-ncci-medically-unlikely-edits
