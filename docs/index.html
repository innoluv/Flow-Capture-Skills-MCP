Flow Code Skills MCP

A modular MCP server that gives Claude Code access to specialized medical coding skills and utility tools

Architecture

skills/ # SKILL.md files (read-only imports)
src/
  index.ts # Local stdio entry point
  server.ts # Shared server factory
  types.ts # SkillModule interface
  skills/ # MCP tool stubs (editable)
    index.ts # Skill registry
    *.skill.ts # One per skill
  lib/ # Shared utilities
orchestrator/ # Claude Code routing skill

Local Setup (5 minutes)

Requires: Node.js 18+, Claude Code CLI, Git

  1. Clone
    git clone https://github.com/innoluv/Flow-Code-Skills-MCP.git
    cd "Flow Code Skills MCP"
  2. Install & build
    npm install && npm run build
  3. Register with Claude Code
    claude mcp add --transport stdio \
      flow-code-skills -- node "$(pwd)/dist/index.js"
  4. Verify — restart Claude Code and ask:
    "List all available flow code skills"

Or one command: bash setup.sh

Available Skills

Medical Coding
em_generateAssigns E&M CPT codes from encounter notes (2021 AMA)
icd_validateValidates ICD-10-CM codes — specificity, excludes, sequencing
medical_necessity_checkChecks ICD-CPT alignment & documentation vs CMS
ncci_edit_checkChecks NCCI PTP edits and MUE compliance
Meta
list_skillsReturns the full catalog of all available skills

Keeping Up to Date

cd "Flow Code Skills MCP"
git pull
npm run build

Then restart your Claude Code session.

Troubleshooting

  • Tools not showing? Run claude mcp list to check registration
  • "Cannot find module"? Run npm run build to recompile
  • Check MCP status inside Claude Code with /mcp
