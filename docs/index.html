Specialized medical coding skills for Claude Code. Set up once, use forever.

Set up once. Use forever. Just chat.

Run the setup once and the skills stay connected permanently — every session, any directory. Claude automatically picks the right skill based on your request. No commands to remember, no manual steps, no code to write. It just works.

⚡ One-Time Setup 5 min

  1. Clone the repo
    $ git clone https://github.com/innoluv/Flow-Code-Skills-MCP.git
    $ cd "Flow Code Skills MCP"
  2. Run setup
    $ bash setup.sh
    Installs deps, builds, and registers with both Claude Code CLI and Claude Desktop automatically.
  3. Restart Claude Code or Claude Desktop
  4. Verify — ask Claude:
    "List all available flow code skills"

Requires: Node.js 18+, Git. Claude Code CLI is optional — setup.sh also configures Claude Desktop directly.

💬 How to Use

Open Claude Code or Claude Desktop (Cowork) and say things like:

"Assign an E&M code for this encounter note"
"Validate ICD codes E11.9 and I10"
"Check NCCI edits for 99214 and 93000"
"Is CPT 99213 medically necessary for E11.9?"

Claude reads your request, calls the right skill, and returns the result — all in the chat.

🔧 Troubleshooting

  • Skills not showing? Check registration with claude mcp list
  • "Cannot find module"? Recompile with npm run build
  • Check MCP status inside any session with /mcp
